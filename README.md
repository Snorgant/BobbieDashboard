# Bobbie Dashboard 🐾

Een persoonlijk, modern dashboard voor Bobbie. Log zijn gewicht, bekijk recente metingen en kies je favoriete kleurthema.

## Functies

- Gewicht loggen en recente metingen bekijken
- Vijf verwisselbare kleurthema's
- Gegevens en themakeuze lokaal opgeslagen
- Responsive ontwerp voor desktop en mobiel
- Eigen geïllustreerd portret van Bobbie

## Lokaal starten

Dubbelklik op `start.bat`, of open `index.html` rechtstreeks in een browser. Er is geen installatie of buildstap nodig.

## Online

De website wordt bij iedere push naar `main` automatisch gepubliceerd met GitHub Pages.
